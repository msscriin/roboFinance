Обновить проект через composer update
Скопировать и переименовать файл .env.example в .env
Сгенерировать ключ через php artisan key:generate
Настроить подключение к Базе данных в файле .env (предварительно необходимо создать базу данных)
Сделать миграцию базы коммандой php artisan migrate
Заполнить базу данных через php artisan db:seed

Маршруты
- GET host.loc/users - возвращает список всех пользователей с балансом
- GET host.loc/users/{id} - возвращает пользователя с балансом по его идентификатору
- GET host.loc/users/{id}/transactions - возвращает список всех переводов в системе совершенных
данным пользователем, с указанием какому пользователю, суммы и даты-времени перевода
- POST host.loc/users/{id}/transactions - совершает перевод денежных средств от пользователя {id}
любому другому пользователю. Информация о получателе и сумме перевода передается в теле
запроса. (переменные в теле запроса idRecipient - ид получателя средств, transfer - сумма перевода)
